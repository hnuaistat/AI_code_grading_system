**업로드할 파일:**
1. **정답 노트북** (`answer.ipynb`)
   - 교수가 작성한 모범 답안
   - 각 셀의 출력값이 기준이 됨

2. **학생 제출물** (`students.zip`)
   ```
   students.zip
   ├── 20210001_홍길동.ipynb
   ├── 20210002_김영희.ipynb
   ├── 20210003_이순신.ipynb
   └── ...
   ```

3. **채점 기준** (`rubric.json`)
   ```json
   {
     "problems": [
       {
         "problem_id": 1,
         "full_score": 20,
         "partial_score_criteria": [
           {"item": "변수명 적절성", "score": 5},
           {"item": "알고리즘 정확성", "score": 10},
           {"item": "출력값 일치", "score": 5}
         ]
       },
       {
         "problem_id": 2,
         "full_score": 30,
         "partial_score_criteria": [
           {"item": "코드 가독성", "score": 10},
           {"item": "효율성", "score": 10},
           {"item": "에러 처리", "score": 10}
         ]
       }
     ]
   }
   ```

## 🎯 주요 기능 요약

| 기능 | 화면 | 설명 |
|------|------|------|
| **로그인** | 화면 1 | JWT 기반 보안 인증 |
| **파일 업로드** | 화면 2 | 3개 파일 드래그&드롭 업로드 |

## 💡 팁

- **채점 중 다른 작업 가능**: 백그라운드에서 처리되므로 페이지를 떠났다가 다시 와도 진행 상황 유지
- **검색 기능**: 학번이나 파일명으로 특정 학생만 빠르게 찾기
- **색상 코드**:
  - 🟢 92% 이상: 우수 (초록색)
  - 🟡 60-92%: 보통 (노랑색)
  - 🔴 60% 미만: 미흡 (빨강색)
- **JSON 형식**: 채점기준 파일은 반드시 UTF-8 인코딩으로 저장
